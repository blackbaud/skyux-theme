The Box component contains content and actions related to one topic.

Use boxes to contain sections in the content areas of action hubs and record pages.

Don't use boxes to contain individual content elements. Use repeaters or other layout approaches instead.

Don't use boxes to contain sections in modal forms or split view workspaces. Use appropriate headers and spacing to separate the content instead.

Don't add boxes to pre-defined page template sections that don't have boxes built in.

Add class="sky-padding-even-xl" to the box to ensure proper spacing of the content from the borders of the box.

Use XL spacing between boxes. You can use fluid grid, CSS Flexbox, CSS Grid or spacing classes depending on your specific scenario.
