The Box component contains content and actions related to one topic.

Use boxes to contain sections in the content areas of action hubs and record pages.

Don't use boxes to contain individual elements. Use repeaters or other layout approaches instead.

Don't use boxes to contain sections in modal forms or split view workspaces. Use headers and spacing to separate the content instead.

Don't add boxes to pre-defined sections when page templates don't use boxes.

Add class="sky-padding-even-xl" to the box to ensure proper spacing of the content from the borders of the box.

Use $sky-theme-modern-space-xl spacing between boxes. Depending on your scenario, you can use fluid grid, CSS Flexbox, CSS Grid or spacing classes.
